Welcome

GPUTOP is a tool to help developers understand GPU performance counters and provide graphical and machine readable data for the performance analysis of drivers and applications. GPUTOP is compatible with all GPU programming apis such as OpenGL, OpenCL or Vulkan.

This is a demo of GPUTOP that lets you explore the interface and some of the metrics supported.

If you want to try out GPUTOP on real hardware please follow these build Instructions and give feedback here.
